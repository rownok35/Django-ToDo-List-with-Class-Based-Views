# Installation

- 1 - create a virtual environment and activate
- 2 - pip install virtualenv
- - virtualenv envname
- - envname\scripts\activate
- 3 - cd into project
- 4 - pip install -r requirements.txt
- 5 - python manage.py runserver

# Features

- User Registration
- Login
- Search
- Full Creat, Read, Update and DELETE functionality.
